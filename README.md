# Ad-Campaign-Dataflow

This project is designed to streamline the process of loading, cleaning, and transforming ad campaign data from a CSV file into a PostgreSQL database. It automates the data ingestion pipeline, ensuring the data is cleaned and ready for analysis or further processing.

Tech Stack: Python, SQL, DBT, Airflow, GCP BiqQuery
